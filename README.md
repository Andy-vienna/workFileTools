Tool-Sammlung zum Einsatz im SINUMERIK Umfeld:
-----------------------------------------------

FileViewer: erkennt in einem Pfad Dateien mit einer bestimmten Endung und öffnet diese Datei im zugeordneten Viewer (hier: gemacht für DDX Viewer (.smo-Files))
FileMover: erkennt bestimmte Dateien in einem Pfad und kopiert diese automatisch in einen Zielpfad
Ordermanager Single: Eröffnet einen Baum aus vorhandenen Dateien (hier: MPF-Dateien), kopiert die auf Doppelklick oder per Button in ein Verzeichnis zur externen Abarbeitung und erzeugt einen Programmaufruf, damit in der SINUMERIK nicht immer ein neues Programm angewählt werden muss.
